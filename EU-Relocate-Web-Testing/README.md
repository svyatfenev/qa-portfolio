# EU-Relocate — тестирование веб-приложения

## 📌 О проекте

**EU-Relocate** — веб-сервис компании, помогающей IT-специалистам с релокацией в страны Европы.

На момент тестирования сервис предоставлял две программы:

- **Базовый пакет** — консультация, подготовка и подача документов, получение визы и ВНЖ.
- **Пакет «Под ключ»** — включает базовый пакет, а также помощь с поиском жилья, открытием банковских счетов, организацией переезда, адаптацией и подачей документов на гражданство.

Проект выполнен в рамках обучения по программе **QA Engineer** и посвящен ручному тестированию пользовательского интерфейса веб-приложения.

---

# 🎯 Цель проекта

Проверить корректность работы пользовательского интерфейса, выявить дефекты и подготовить тестовую документацию.

# ✅ Выполненные задачи

## Тестирование главной страницы

Проведено тестирование новых пользовательских форм:

- «Бесплатный чек-лист по переезду»
- «Получите бесплатную консультацию»

В рамках работы были выполнены:

- анализ требований;
- разработка тест-кейсов;
- функциональное тестирование;
- оформление баг-репортов.

---

## Регрессионное и повторное тестирование

После исправления найденных дефектов были проведены:

- повторное тестирование (Re-test);
- регрессионное тестирование;
- актуализация тестовой документации;
- проверка исправленных дефектов.
